#Calculadora Avançada na Linguagem C!

Um projeto básico apenas para testes, funciona apenas no Windows devido ao comando system("pause").
Feito na linguagem C utilizando o VsCode.  
Realiza calculos de: adição, subtração, multiplição, divisão, conversor de temperaturas (Celsius e Fahrenheit) e calculo de medidas (area,perimetro e volume).   
Para visualizar o codigo fonte, verifique o arquivo "multi.c".
